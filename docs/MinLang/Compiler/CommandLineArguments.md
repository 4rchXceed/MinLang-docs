# Command Line Arguments

```bash
minlang <source_file> <output_file> <x>,<y>,<z>
```

- `<source_file>`: The path to the MinLang source file you want to compile.
- `<output_file>`: The desired path for the compiled output file.
- `<x>,<y>,<z>`: Three comma-separated coordinates that represent the target location for the compiled program to operate within. (Minecraft world coordinates)
